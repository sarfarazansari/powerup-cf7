A trustworthy post data plugin for Contact Form 7.

== Description ==

Powerup CF7 is a post data plugin originally created for [Contact Form 7](https://wordpress.org/plugins/contact-form-7/), which doesn't Post submitted messages to a URL.

After activation of the plugin, you'll find *Powerup CF7* inside the WordPress admin Setting menu. All messages through contact forms are posted on your configure URL.

For more detailed information, please refer to the [Powerup CF7](https://viasocket.com/).

== Installation ==

1. Upload the entire `Powerup CF7` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
